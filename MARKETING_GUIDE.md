## 앱 개요

**앱 이름:** Mandarat  
**버전:** 1.0.0  
**카테고리:** 생산성, 목표관리, 플래너  
**핵심 콘셉트:** 오타니 쇼헤이의 만다라트 전략을 디지털화한 목표 설계 및 실행 앱

### 한 줄 설명
"큰 꿈을 64개의 실행 가능한 목표로 변환하고, 매일 그것을 이루어가는 앱"

## 타겟 오디언스

### 주 타겟 (Primary)
- **연령:** 20-40대
- **특성:** 
  - 명확한 목표를 가지고 있으나 실행에 어려움을 겪는 사람들
  - 자기계발에 관심이 높은 사람들
  - 계획은 세우지만 실천하지 못한 경험이 있는 사람들
  - 오타니 쇼헤이의 팬/스토리에 공감하는 사람들

### 부 타겟 (Secondary)
- 학생 (대학생, 입시생)
- 직장인 (커리어 개발 목표)
- 창업가 (사업 목표)
- 운동선수 (성취 목표)

## 핵심 가치 제안

### USP (Unique Selling Proposition)
1. **검증된 전략:** 오타니 쇼헤이가 메이저리그 입성부터 MVP 수상까지 실제로 사용한 만다라트 전략
2. **과학적 효과:** 목표를 기록하면 달성확률 42% 상승, 시각화하면 73% 상승
3. **실행 중심:** 단순 기록이 아닌 "매일의 선택"을 가이드하는 실행 도구
4. **심플함:** 복잡한 기능이 아닌 64칸의 단순하고 명확한 구조

### 주요 이점 (Benefits)
| 기능 | 이점 | 효과 |
|------|------|------|
| 꿈을 설계하다 | 추상적 목표가 구체화됨 | 방향 감각 회복 |
| 매일 실행하다 | 매 순간의 선택이 명확해짐 | 실행력 강화 |
| 진행을 확인하다 | 성장이 눈에 띄어짐 | 동기 부여 지속 |

## 기능 설명

### 버전별 기능 설명

#### 🎯 30초 버전 (광고용)
```
Mandarat은 목표 달성 앱입니다.
큰 꿈을 64개의 작은 목표로 분해하고,
매일 그것을 이루어가면서 
성장을 확인할 수 있습니다.

오타니처럼 꿈꾸고, Mandarat으로 이루세요.
```

#### 📱 1분 버전 (상세 설명)
```
**Mandarat은 3가지를 합니다:**

1️⃣ 꿈을 설계하다
   큰 꿈을 64개의 작은 목표로 분해합니다.
   방향을 잃지 않게 해줍니다.

2️⃣ 매일 실행하다
   매 순간의 선택을 명확하게 가이드합니다.
   실행이 쉬워집니다.

3️⃣ 진행을 확인하다
   목표 달성의 모든 순간을 시각화합니다.
   동기가 유지됩니다.

오타니 쇼헤이가 고교생 때 했던 것처럼,
당신의 목표를 설계하고 이루세요.
```

#### 📊 심화 버전 (문서/웹용)
```
**Mandarat: 목표를 현실로 만드는 64칸의 여정**

메이저리그 MVP 오타니 쇼헤이는 고교생 시절,
단 한 장의 종이에 만다라트을 그렸습니다.

그 종이에는:
- 중심: 최종 목표 (메이저리그 입성)
- 바깥쪽: 그걸 이루기 위한 8개 테마
- 각 테마의 8개 하위 목표 = 총 64개

이 전략은 실제로 그를 MVP까지 만들었습니다.

**이제 Mandarat이 당신의 종이가 되어줍니다.**

✅ 목표를 기록하면 42% 더 이루기 쉬워집니다
✅ 목표를 시각화하면 73% 더 이루기 쉬워집니다
✅ Mandarat은 둘 다 합친 앱입니다

당신의 꿈은?
```

## 광고 메시지 가이드

### 핵심 메시지 3가지

#### 메시지 1: 검증된 전략 (신뢰성)
```
"메이저리그 MVP가 실제로 사용한 전략"
- 오타니의 성공 사례
- 검증된 효과
- 실제 이룬 사람의 도구
```

#### 메시지 2: 실행 중심 (차별성)
```
"기록이 아닌 실행의 도구"
- 단순 플래너가 아님
- 매일의 선택을 가이드
- 결과가 나오는 앱
```

#### 메시지 3: 감정적 공감 (감성)
```
"목표가 흐려지는 경험을 끝내세요"
- 누구나 겪는 문제
- 명확한 해결
- 다시 시작할 용기
```

### ✅ 반드시 포함해야 할 요소
