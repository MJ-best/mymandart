# 만다라트 저니 테스트 가이드

## 🧪 테스트 시나리오

### 1단계: 목표 정의 테스트
1. **템플릿 선택 테스트**
   - 페이지 로드 시 템플릿 버튼들이 표시되는지 확인
   - 템플릿 클릭 시 텍스트 영역에 자동 입력되는지 확인

2. **키워드 추천 테스트**
   - 키워드 칩들이 카테고리별로 색상 구분되어 표시되는지 확인
   - 키워드 클릭 시 텍스트 영역에 추가되는지 확인
   - 툴팁 호버 시 설명이 표시되는지 확인

3. **입력 검증 테스트**
   - 10자 미만 입력 시 경고 메시지 표시 확인
   - 10자 이상 입력 시 성공 메시지 표시 확인
   - "다음" 버튼 활성화/비활성화 확인

### 2단계: 테마 설정 테스트
1. **테마 입력 테스트**
   - 8개 테마 입력 필드가 표시되는지 확인
   - 각 테마별로 키워드 추천이 작동하는지 확인
   - 실시간 저장이 되는지 확인

2. **진행 상황 표시 테스트**
   - 오른쪽에 진행 상황 패널이 표시되는지 확인
   - 테마 완성도가 실시간으로 업데이트되는지 확인
   - 모든 테마 입력 시 "다음" 버튼 활성화 확인

### 3단계: 액션 아이템 테스트
1. **테마별 탭 테스트**
   - 8개 테마 탭이 표시되는지 확인
   - 탭 전환 시 해당 테마의 액션 아이템이 표시되는지 확인

2. **액션 아이템 입력 테스트**
   - 각 테마별로 8개 액션 아이템 입력 필드 확인
   - 체크박스로 완료 상태 관리 확인
   - 키워드 추천이 현재 활성 테마에 적용되는지 확인

3. **진행 상황 업데이트 테스트**
   - 액션 아이템 완료 시 진행 상황이 업데이트되는지 확인
   - 전체 완성도가 정확히 계산되는지 확인

### 만다라트 보기 테스트
1. **그리드 표시 테스트**
   - 9x9 그리드가 올바르게 표시되는지 확인
   - 중앙에 목표, 주변에 테마, 외곽에 액션 아이템 배치 확인
   - 완료된 액션 아이템이 다른 색상으로 표시되는지 확인

2. **상호작용 테스트**
   - 테마 클릭 시 상세 보기로 전환되는지 확인
   - "전체보기" 버튼으로 돌아가는지 확인

3. **내보내기 테스트**
   - 이미지 내보내기 기능 작동 확인
   - JSON 내보내기 기능 작동 확인
   - 파일 다운로드가 정상적으로 되는지 확인
